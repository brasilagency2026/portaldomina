1. VISÃO GERAL DO PROJETO
Desenvolver um portal web geolocalizado em português brasileiro para profissionais dominatrixes, com sistema de cadastro verificado, geolocalização, pagamentos múltiplos (cripto BRZ, PIX, cartão) e painel administrativo.
2. FUNCIONALIDADES PRINCIPAIS
2.1 PORTAL PÚBLICO (Front-end)

Idioma: Português do Brasil (pt-BR)
Mapa interativo: Integração Google Maps API mostrando dominatrixes por geolocalização
Filtros de busca:

Raio de distância (5km, 10km, 20km, 50km)
Especialidades/práticas
Disponibilidade
Tipo de atendimento (local próprio, hotel, domicílio)
Plano (gratuito vs premium)



2.2 PERFIS DAS DOMINATRIXES
Plano Gratuito inclui:

Nome artístico/profissional
Galeria de até 5 fotos
Descrição/bio (máximo 500 caracteres)
Especialidades (tags)
Endereço (cidade/bairro - sem rua específica)
Botão WhatsApp (link direto: https://wa.me/55[número])
Botão Waze (link direto para navegação: https://waze.com/ul?ll=[latitude],[longitude]&navigate=yes)
Endereços de carteiras cripto para receber pagamentos de clientes:

BRZ (Brazilian Digital Token) - via Chainless
Bitcoin (BTC)
Ethereum (ETH)
USDT (TRC20/ERC20)
PIX (chave PIX)
Cartão de crédito (link de pagamento personalizado)



Plano Premium adiciona:

Galeria ilimitada de fotos
Upload de até 3 vídeos (máximo 2 minutos cada)
Destaque no mapa dentro do raio de 20km (pin diferenciado, cor especial, aparece primeiro)
Aparece prioritariamente nas buscas gerais
Endereço completo com Google Maps embed
Badge "Premium" no perfil
Estatísticas de visualizações
Valor: R$ 49,90/mês

4. SISTEMA DE PAGAMENTO DA ASSINATURA PREMIUM
Opções de pagamento para assinatura Premium (R$ 49,90/mês):
Opção 1 - Criptomoeda BRZ via Chainless:

Integração com Chainless API
Pagamento em BRZ (Brazilian Digital Token)
Conversão automática do valor em BRL para BRZ
Confirmação automática após blockchain confirmation
Link de pagamento gerado automaticamente

Opção 2 - PIX:

Integração com Mercado Pago API
Geração de QR Code PIX
Confirmação em tempo real
Chave PIX da plataforma

Opção 3 - Cartão de Crédito:

Integração com Mercado Pago
Aceita: Visa, Mastercard, Elo, Amex
Parcelamento em até 3x sem juros (opcional)
Pagamento recorrente automático (opcional)
Tokenização segura de cartões

Features do sistema de pagamento:

Múltiplas tentativas de pagamento
Notificações de vencimento:

7 dias antes
3 dias antes
1 dia antes
No dia do vencimento


Período de grace (3 dias após vencimento)
Downgrade automático para plano gratuito após grace period
Histórico completo de transações
Comprovantes em PDF

8. FEATURES DE GEOLOCALIZAÇÃO
Sistema de destaque Premium (20km):

Quando usuário busca no mapa, perfis Premium aparecem:

Com pin diferenciado (cor dourada/premium)
Maior no mapa
Sempre no topo da lista lateral
Dentro do raio de 20km da localização do usuário


Algoritmo de priorização:

Perfis Premium dentro de 20km
Perfis gratuitos mais próximos
Todos os demais por distância



Permissões de localização:

Solicitar localização do usuário (geolocation API)
Fallback para busca por CEP/cidade
Salvamento de última localização (cookie/localStorage)
